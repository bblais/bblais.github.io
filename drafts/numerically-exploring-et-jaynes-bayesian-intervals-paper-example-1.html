"Numerically exploring ET Jaynes Bayesian Intervals Paper: Example 1"

Example 1

Mon 11 August 2025 In #math

In E T Jaynes's paper Confidence vs Bayesian Intervals is one of my favorite papers of all time. There are a number of very specific examples where he shows differences between common frequentist practice and Bayesian analysis. In this series I hope to go through all the examples from the paper, but done numerically.

The problem

Frequentist approach

# ChatGPT-5 Aug 11 2025
import scipy.stats as stats

# Data
nA, sA = 9, 7.48
nB, sB = 4, 6.48

# Variances
varA = sA**2
varB = sB**2

# Ensure the larger variance is in numerator
if varA >= varB:
    F = varA / varB
    dfn, dfd = nA - 1, nB - 1
else:
    F = varB / varA
    dfn, dfd = nB - 1, nA - 1

alpha = 0.05
F_crit_upper = stats.f.ppf(1 - alpha/2, dfn, dfd)
F_crit_lower = stats.f.ppf(alpha/2, dfn, dfd)

p_value = 2 * min(stats.f.cdf(F, dfn, dfd), 1 - stats.f.cdf(F, dfn, dfd))

F statistic = 1.332
dfn=8, dfd=3
F critical lower = 0.185, upper = 14.540
p-value = 0.8959

ChatGPT-5 Aug 11 2025: We fail to reject the null hypothesis that the variances are equal — there is no significant difference in the standard deviations at the 95% confidence level.

import scipy.stats as stats
import numpy as np

# Data
nA, meanA, sA = 9, 42, 7.48
nB, meanB, sB = 4, 50, 6.48

# Pooled variance
sp2 = ((nA - 1) * sA**2 + (nB - 1) * sB**2) / (nA + nB - 2)
sp = np.sqrt(sp2)

# t-statistic
t_stat = (meanA - meanB) / (sp * np.sqrt(1/nA + 1/nB))
df = nA + nB - 2

# p-value
p_value = 2 * stats.t.cdf(t_stat, df) if t_stat < 0 else 2 * (1 - stats.t.cdf(t_stat, df))

print(f"Pooled s = {sp:.3f}")
print(f"t = {t_stat:.3f}, df = {df}")
print(f"p-value = {p_value:.4f}")

Pooled s = 7.221
t = -1.844, df = 11
p-value = 0.0923

ChatGPT Aug 11 2025: We fail to reject \(H_0\): there is no statistically significant difference between the mean lifetimes.
